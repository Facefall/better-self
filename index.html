2023年了，不知不觉已经在字节工作一年多，快两年了。但自己只是一个初级前端开发工程师， 刚刚工作的时候，由于业务上啥都不懂，每天都能学到新知识，一干就是一整天。

而现在，日常的工作对我个人而言没有挑战，只专注于业务，成长很小。 我想成为资深的开发工程师，于是决定建立自己的个人博客，监督自己。

给自己设立几个小目标：

  1. 记录如何部署 github pages
  2. 将文档补全 ts + react
  3. 记录业务知识

要紧的是果敢的迈出第一步，对与错都先不管。自古就没有先设计好再开步的事，别想把一切都弄清楚再去走路。鲁莽者要学会思考，善思者要克服的是犹豫。目的可求完美，举步之际则无需周全。

我发现， 理智的做法只有在做很小的决定时才有效。 至于改变人生的事情，你必须冒险。—要有理智的冒险
