# Prerequisites

你可以使用本册子查询任何 level 的技巧，但本使用手册假设你有着对 react 和 Typescript 最基本的了解。 以下是一些前置条件：

  • 对 React 有基本的了解
  • 熟悉 Typescript 基本内容和常见类型
  • 在本手册，我们假设你正在使用最新版本的 react 和 Typesript

# React and TypeScript starter kits

React 已经为读者提供了一些从流行框架生成新 react 项目的方式

Here's how to start them with TypeScript:

  • Next.js (opens new window): npx create-next-app@latest --ts
  • Remix (opens new window): npx create-remix@latest
  • Gatsby (opens new window): npm init gatsby --ts
  • Expo (opens new window): npx create-react-native-app -t with-typescript
